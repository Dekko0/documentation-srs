Requisitos Não Funcionais (RNF)

RNF01

Desempenho

Tempo de resposta de até 3 segundos para qualquer operação do sistema.

RNF02

Autenticação

Login integrado com autenticação institucional (SSO/LDAP).

RNF03

Disponibilidade

Disponibilidade mínima de 99% em horário comercial.

RNF04

Segurança

Criptografia de dados sensíveis e backup automático diário.

RNF05

Usabilidade

Interface intuitiva e de fácil uso para todos os perfis de usuários.
